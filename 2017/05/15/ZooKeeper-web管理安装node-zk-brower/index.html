ZooKeeper web管理安装node-zk-brower

Veröffentlicht am 2017-05-15

node-zk-browser是zookeeper web管理工具,使用Node.js,基于express.js和node-zookeeper开发,以树的形式展示zookeeper的数据,登录后可以对节点数据进行新增,编辑,删除.
案装配置流程如下:
1.安装node.js
wget https://github.com/joyent/node/archive/v0.10.35.tar.gz
[root@DB zookeeper-web安装包]# wget https://github.com/joyent/node/archive/v0.10.35.tar.gz
[root@DB zookeeper-web安装包]# ls
v0.10.35.tar.gz
安装:
[root@DB zookeeper-web安装包]# wget https://github.com/joyent/node/archive/v0.10.35.tar.gz
[root@DB zookeeper-web安装包]# ls
v0.10.35.tar.gz
安装:注意需要gcc-c++支持,若提示相应错误,需安装yum -y install gcc-c++
[root@DB zookeeper-web安装包]# tar -xzvf v0.10.35.tar.gz
[root@DB node-v0.x-archive-0.10.35]# ./configure –prefix=/data/apps/nodejs
[root@DB node-v0.x-archive-0.10.35]# make && make install
[root@DB zookeeper-web安装包]# tar -xzvf v0.10.35.tar.gz
[root@DB node-v0.x-archive-0.10.35]# ./configure –prefix=/data/apps/nodesj
[root@DB node-v0.x-archive-0.10.35]# make && make install
配置环境变量:~/.bash_profile
export PATH=$PATH:/data/apps/nodejs/bin
export NODE_PATH=/data/apps/nodejs:/data/apps/nodejs/lib/node_modules
这个版本的nodesjs是自带npm
运行下面的脚本hi_cl.js
[root@DB ~]# cat hi_cl.js
var http = require(“http”);
http.createServer(function(request, response) {
response.writeHead(200,{“Content-Type”:”text/plain”});
response.write(“wo shi cl”);
response.end();
}).listen(5000);
注意防火墙！
[root@DB ~]# node hi_cl.js
查看效果!
[root@ok caffe-linux]# curl http://10.100.0.220:5000
wo shi cl
安装信赖包:
https://github.com/killme2008/node-zk-browser
[root@DB zookeeper-web安装包]# ls
node-v0.x-archive-0.10.35 node-zk-browser-master.zip v0.10.35.tar.gz
[root@DB zookeeper-web安装包]# unzip node-zk-browser-master.zip
[root@DB zookeeper-web安装包]# ls
node-v0.x-archive-0.10.35 node-zk-browser-master node-zk-browser-master.zip v0.10.35.tar.gz
[root@DB zookeeper-web安装包]# mv node-zk-browser-master /data/apps/
[root@DB apps]# cd node-zk-browser-master
[root@DB node-zk-browser-master]# ls
app.js LICENSE.txt logs package.json public README.md start.sh test user.json views zk.js
进入node-zk-brower-master目录(重点注意一定要进入这个目录下)执行下列命令来安装信赖包(ejs,express,express-namespace和zookeeper)
如下列:
npm install -d ejs
[root@DB node-zk-browser-master]# npm install -d ejs
npm info it worked if it ends with ok

同上安装下面包:
npm install -d express
npm install -d express-namespace
npm install -d zookeeper
注意!!!!!!
安装完成后会在node-zk-browser下生成ejs,express,express-namespace,zookeeper
[root@DB node-zk-browser-master]# ls node_modules/
ejs express express-namespace zookeeper
配置node-zk-browser-master
修改启动脚本配置zk地址:start.sh
[root@DB node-zk-browser-master]# cat start.sh
LOGFILE=$(dirname $0)/logs/node-zk-browser.log
export ZK_HOST=”10.100.0.220:2181,182.92.189.44:2181,182.91.22.35:2181”
nohup node $(dirname $0)/app.js 2>&1 >>$LOGFILE &

启动服务:
